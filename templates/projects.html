C2P

A chrome extension for making practice easy in codechef.
In Codechef, if you want to practice previous contest, then you can't submit the solutions directly from contest page.
You have to search for the problem in the problem archive, which is quite uncomfortable and uneasy.

This extension helps you out here - once you are in the contest problem page and you want to submit solution for the problem, just click on the extension icon and then click "go to practice page". And You can now submit the solution! :) The code can be found here :
How to use :
1)Go to "chrome://extension" (Type in the browser without quotes)
2)tick the "developer mode"
3)Load the folder of C2P either from github or the link provided below. ("Load unpacked extension").
4)You will see a new icon in the toolbar!
5) Go to any previous practice contest and open a problem and click on the new icon of c2p and keep solving.
