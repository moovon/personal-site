HOW TO INSTALL UBUNTU USING BOOTABLE PEN-DRIVE!
29th December'2015

1)You need to have a pendrive of size 4GB or more.

2) Download Ubuntu desktop .iso file from ubuntu.org for free.

3) Download and install universal usb installer from << http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/ >> .

4)Run the usb installer and pick up the version of the ubuntu you downloaded.

5) Then select up the .iso file

6) click CREATE

now your bootable pendrive is ready!

7) Now restart your computer and press f1 or del (or whatever ) to get into the boot screen.

8) Then give the highest priority to your pendrive in the “harddisks” (pendrive must be plugged in).

9)Then save and exit!

10) A ubuntu window will appear and then follow the procedure to get ubuntu installed!

11) ENJOY UBUNTU!
